# Caso de Uso 06

## Titulo
Encerrar Solicitação do Cliente

## Descrição
Este caso de uso descreve o processo pelo qual um funcionário da Dosimagem encerra uma solicitação de serviço do cliente após o processamento ter sido concluído.

## Ator Principal
Funcionário da Dosimagem

## Pré-condições
Orelatório foi anexado à solicitação do cliente no sistema. O funcionário selecionou uma solicitação de serviço pendente para processamento e concluiu todas as etapas necessárias.

## Fluxo Principal
1.	O caso de uso começa quando o funcionário finaliza o processamento da solicitação de serviço do cliente, incluindo a anexação do relatório.
2.	O funcionário acessa a solicitação de serviço pendente selecionada no sistema.
3.	O sistema fornece uma opção para encerrar a solicitação do cliente.
4.	O funcionário seleciona a opção para encerrar a solicitação.
5.	O sistema confirma o encerramento bem-sucedido da solicitação e atualiza o status da mesma para “Encerrada”.

## Pós-condições
A solicitação de serviço do cliente está encerrada no sistema da Dosimagem, indicando que o processo foi concluído e todas as etapas necessárias foram realizadas.
